# Dart Course: Learning & Practice

這個倉庫記錄了我學習 Dart 程式語言的完整歷程。所有的課程內容與範例皆源自於 **Andrea Bizzotto** 所開設的優質課程。

## 📖 關於本專案
本專案的主要目的是將學習過程中的筆記、程式碼練習以及專案挑戰進行系統化整理。透過實踐 Andrea Bizzotto 的教學內容，深入掌握 Dart 的核心語法與進階特性。

## 🏗️ 學習內容
* **Dart 核心語法**：變數、型別系統、控制流。
* **函數式編程**：匿名函數、集合操作 (Map/List/Set)。
* **物件導向 (OOP)**：類別、繼承、Mixins 與介面。
* **非同步程式設計**：Future、Stream 與非同步處理。
* **錯誤處理**：例外排除與程式碼健全度優化。
